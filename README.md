# Auto Submitter

Automated form and comment submission system with campaign management, web crawling, and Playwright automation.

## Features
- Campaign management dashboard
- Automated form detection and submission
- Comment posting automation
- Contact extraction (email & phone)
- Excel exports
- Test mode available at `/test`
